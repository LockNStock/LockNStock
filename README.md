## High-Impact Automation Pipelines

### [short-drama-forecaster](https://github.com/LockNStock/short-drama-forecaster)
> **Short-Drama Views & LTV Predictive Forecaster**
*   **The Problem**: Making dynamic ad-pacing and content investment valuations manually for hundreds of early-run micro-dramas is slow and burns massive paid media budgets on high-decay flops.
*   **The Breakthrough**: Engineered an analytical machine learning pipeline. Fits viewership decay curves using non-linear least squares optimization (SciPy) over early free episodes, training a Random Forest Regressor to forecast locked-episode premium paywall unlocks and overall campaign ROI.
*   *Tech stack: Python 3, SciPy Optimize, Scikit-Learn, Pandas & NumPy.*

### [ai-shortdrama-pipeline](https://github.com/LockNStock/ai-shortdrama-pipeline)
> **Short Drama Generative Pipeline**
*   **The Problem**: Production timelines for micro-dramas are severely bottlenecked by manual voiceover recording, millisecond subtitle syncing, and character video assembly.
*   **The Breakthrough**: Constructed an end-to-end media compiler that parses screenplays, locks character-consistent neural voice seeds (via local Kokoro MLX & ChatTTS), auto-calculates millisecond SRT subtitle tracks, and multiplexes them with consistent vertical clips via Stable Video Diffusion (ComfyUI).
*   *Tech stack: Kokoro MLX, ChatTTS, ComfyUI (Stable Video Diffusion), FFmpeg Shell.*

### [yt-claims-dashboard](https://github.com/LockNStock/yt-claims-dashboard)
> **YT-Monitor: Copyright Claims & Rights Auditor**
*   **The Problem**: Managing millions of daily copyright claim logs from multiple regional platforms suffers from duplicate entries and timezone offsets, risking missing high-value IP assets.
*   **The Breakthrough**: Designed a high-throughput Pandas pipeline to merge and clean heterogeneous CSV inputs, dropping duplicates, mapping RightsID compliance schemas, and isolating high-priority copyright targets (e.g. *Rick and Morty*).
*   *Tech stack: Python, Pandas, NumPy, SQLite 3.*

### [career-scout-rpa-suite](https://github.com/LockNStock/career-scout-rpa-suite)
> **CareerScout: AI Resume Tailoring & RPA Extension Suite**
*   **The Problem**: Job hunting forms are repetitive, and modern reactive frameworks (React/Vue) clear inputs populated via simple DOM setters upon submission.
*   **The Breakthrough**: Built a collaborative job-search engineering suite. Features a backend Python-based vector matcher (double-bag-of-words keyword weighting) that ranks master-profile bullet points dynamically, coupled with a native Chrome MV3 unpacked browser extension that intercepts DOM input prototypes to bypass React view-state sync constraints and auto-submits customized recruitment form queries.
*   *Tech stack: Chrome MV3 API, Google Gemini API, Javascript (ES6), Python 3.*

### [ChaosGuard](https://github.com/LockNStock/ChaosGuard)
> **ChaosGuard: Adaptive Bayesian Scheduler (v9.0 Flagship)**
*   **The Problem**: Web-based automation agents (like those interacting with Gemini or recruitment portals) are highly susceptible to behavioral analysis and rate-limiting if they follow rigid, predictable task schedules.
*   **The Breakthrough**: Engineered an environment-aware scheduling engine. It utilizes a **10,000 Prime Domain (1,229 seeds)** to eliminate statistical frequency peaks, coupled with a **Bayesian Feedback Loop** that "senses" server latency. Features **Dual-Profile Interaction (Smooth vs. High-Contrast)** to mimic specific human cognitive states during long-form Web/LLM sessions.
*   *Tech stack: Python 3, Bayesian Inference, Inhomogeneous Poisson Process (IPP), Chaos Theory.*
